# 🔐 Login System

Sistema de autenticação simples desenvolvido com foco em aprendizado e boas práticas de desenvolvimento.

## 📌 Sobre o projeto

Este projeto implementa um sistema de login com funcionalidades básicas de autenticação de usuários, podendo incluir validação de credenciais, armazenamento de dados e controle de acesso.

O objetivo é aplicar conceitos de desenvolvimento, organização de código e segurança básica em aplicações.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Login com validação de credenciais
- Controle de sessão
- (Opcional) Persistência de dados
- (Opcional) Tratamento de erros
